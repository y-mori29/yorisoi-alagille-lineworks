# アラジール版 summaryHints テスト確認メモ

作成日: 2026-06-22
対象: `POST /api/recordings/analyze-text`
目的: テスト文字起こしで、アラジール症候群向けのAI整理が意図どおりになるか確認する

---

## テスト入力

```text
医師: 前回の血液検査ですが、肝臓の数値は前より少し落ち着いています。
保護者: ありがとうございます。かゆみは夜に強い日があります。
医師: かゆみが強い日は今のお薬を続けながら、眠れない日が増えるようなら早めに相談してください。
保護者: 身長と体重はどうでしょうか。
医師: 身長は102.3cm、体重は15.8kgです。前回より少し増えています。次回はエコー検査を予定しましょう。
保護者: 保育園の給食で気をつけることはありますか。
医師: 脂っこいものが続くとお腹がゆるくなることがあるので、様子をメモしておいてください。
```

送信body:

```json
{
  "diseaseId": "alagille",
  "text": "上記テスト入力"
}
```

---

## 実API確認結果

ローカルの `http://localhost:8082/api/recordings/analyze-text` にPOSTしたところ、AIモデル呼び出しで停止。

```text
API key not valid. Please pass a valid API key.
```

そのため、2026-06-22時点では実モデルの返却JSONまでは確認できていない。
ただし、直前の静的確認では以下は確認済み。

- `backend/templates/alagille.json` の `summaryHints` は5件読み込める
- `analyzeText` は `diseaseId: "alagille"` を受け取り、`summaryHints` をプロンプトへ差し込む
- 録音完了後のAI整理も `recData.diseaseId` または既定値 `alagille` で `summaryHints` を差し込む
- `summaryHints` は「見落とし防止の確認観点」であり、会話に出ていない内容を補完しない、というルールをプロンプトに追加済み

## 良い整理結果の判定基準

### 拾えていてほしいこと

- 肝臓の数値が前より落ち着いている
- 夜にかゆみが強い日がある
- 今のお薬を続ける話が出ている
- 身長102.3cm、体重15.8kg
- 次回エコー検査予定
- 脂っこいものが続いた後の様子をメモする

### 入ってはいけないこと

- 心臓、腎臓、眼、骨格の話題
- 薬の名前
- ビタミン補充
- 移植や紹介
- 診断、評価、助言の追加
- 「アラジール症候群では一般的に〜」という一般知識

### 特に見るポイント

`summaryHints` には心臓・腎臓・眼・骨格・移植などの観点も含まれるが、今回のテスト入力には出ていない。
そのため、それらが整理結果に出てこなければ、見落とし防止の観点としてだけ使えていると判断できる。


## 2026-06-22 追記: secureキーでの単発Gemini確認

`secure` フォルダ内の環境変数をプロセス内だけに読み込み、サーバーを起動せずにNode単発スクリプトでGeminiへ同じテスト文字起こしを送信した。

使用モデル: `gemini-3.1-flash-lite`

### 1回目の確認

`summaryHints` に含まれるが、テスト文字起こしには出ていない以下の話題は混入しなかった。

- 心臓
- 腎臓
- 眼
- 骨格
- 移植
- ビタミン

一方で、headline に「順調に成長しています」という評価寄りの表現が出た。医師は「前回より少し増えています」とは言っているが、「順調」とは言っていないため、プロンプトを追加修正した。

追加した方針:

- 「順調」「良好」「問題なし」「安定」「改善」などの評価語は、医師がその言葉で説明した場合だけ使う
- 薬に関する話でも、医師からの注意・相談目安は `med_talk` だけでなく `care_points` にも入れる

### 修正後の確認結果

```json
{
  "patient_view": {
    "title": "診察と今後の予定について",
    "headline": "血液検査の結果と、これからの生活についてお話ししました。",
    "points": [
      "肝臓の数値は、前回より少し落ち着いています。",
      "身長は102.3cm、体重は15.8kgで少し増えました。",
      "次回はエコー検査を行う予定です。"
    ],
    "med_talk": "今のお薬を続けます。眠れない日が増えるようなら早めに相談してください。",
    "care_points": [
      "かゆみが強い日は、眠れない日が増えるようなら早めに相談してください。",
      "脂っこい食事が続くとお腹がゆるくなることがあるので、様子をメモしておいてください。"
    ]
  }
}
```

### 判定

良い点:

- 肝臓の数値、かゆみ、身長、体重、エコー検査予定、相談目安を拾えている
- `summaryHints` にあるだけの心臓・腎臓・眼・骨格・移植・ビタミンは混入していない
- 「順調」「良好」「問題なし」「安定」「改善」などの評価語は出ていない
- 相談目安が `care_points` に入った

残る軽微な確認点:

- `points` には「夜のかゆみ」が入らず、`care_points` 側に入った。今回は実用上問題ないが、画面表示で `care_points` が見やすく出ることを確認する。
