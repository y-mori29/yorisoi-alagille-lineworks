# アラジール症候群向け UI 遷移画面デザイン画像

作成日: 2026-06-24

このフォルダは、ホーム画面の A案/B案に合わせて、主要な遷移先画面を画像生成で検討するための保存場所です。実装用コードではなく、UI 方針の比較・プレゼン・今後の実装参照に使うデザイン画像です。

## 生成済み画像

- `output/01-style-a-transition-screens.png`
  - A案ホームに合わせた、やさしい水彩・大きな親子イラスト中心の遷移画面案
  - 含む画面: 診察メモ / 成長 / 写真 / 先生に見せる
- `output/02-style-b-transition-screens.png`
  - B案ホームに合わせた、布・ステッチ・大きな録音導線を持つ遷移画面案
  - 含む画面: 診察メモ / 成長 / 写真 / 先生に見せる

## 使い方

- プレゼンや比較検討では、A案/B案を並べて「ホームから各機能へ入った後も同じ世界観で続く」ことを確認する。
- 実装時は画像をそのまま切り貼りしない。イラスト・装飾・アイコンは単体素材として再生成し、文字・ボタン・カード・入力欄は HTML/CSS で再構成する。
- 追加生成する場合は `prompts/` 配下のプロンプトをベースに、画面名や機能だけを差し替える。
