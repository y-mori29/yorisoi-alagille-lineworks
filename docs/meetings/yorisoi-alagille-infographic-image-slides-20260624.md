## 目的

完成済みのインフォグラフィック画像を、Googleスライドの各ページに1枚ずつ全画面配置した資料。
説明用のテキストボックスや追加レイアウトは入れず、画像スライドとしてそのままミーティングで見せる。

3. 機能2｜成長のきろく
4. 機能3｜写真アルバム
5. 機能4｜先生に見せる
